Интернет-магазин электроники

Разработка современного интернет-магазина с каталогом на 5000+ товаров и интеграцией с 1С

Клиент ElectroTech
Год 2024
Категория E-commerce
Технологии React, Node.js, MongoDB

Задача

Клиент — розничная сеть по продаже электроники, имеющая несколько физических магазинов. Основные задачи:

  • Создать современный интернет-магазин для расширения клиентской базы
  • Реализовать синхронизацию товаров с системой 1С
  • Интегрировать несколько способов оплаты и доставки
  • Обеспечить быструю загрузку страниц при большом каталоге
  • Создать удобную систему фильтрации и поиска

Процесс работы

Проект реализовывался в несколько этапов:

1. Анализ и проектирование

Провели анализ конкурентов, определили целевую аудиторию и разработали пользовательские сценарии.

2. Дизайн и прототипирование

Создали дизайн-систему и интерактивные прототипы основных страниц: главная, каталог, карточка товара, корзина.

3. Разработка

Фронтенд на React с использованием Next.js для SSR. Бэкенд на Node.js с REST API. База данных MongoDB.

4. Интеграции

Интегрировали системы: 1С для товаров, Яндекс.Касса для оплаты, СДЭК для доставки, amoCRM для заявок.

5. Тестирование и запуск

Провели нагрузочное тестирование, протестировали на разных устройствах и запустили продакшен.
